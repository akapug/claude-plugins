## Quick Reference

### Running Code
```bash
elide run script.ts      # Run TypeScript
elide run script.py      # Run Python (3x faster!)
elide run script.rb      # Run Ruby (experimental)
elide                    # Interactive REPL
```

### Cross-Language Imports
```typescript
// TypeScript importing Python
import py from "./module.py"
const result = py.process(data)
```

```python
# Python importing TypeScript
import js from "./utils.ts"
formatted = js.format_output(raw)
```
